✷ من نحن ؟

طمسة هو موقع لفعالية مخصصة للرسامين، خصيصًا الطامسين. حيث يتم تقسيمنا إلى فريقين ( أو ثلاثة ) لنتقاتل بالفن. يهدف إلى لم شمل الطامسين ورفع روحهم الرياضية وتطوير مهاراتهم!

وعن من هي طمسة كـ شخص؟ هذه مُرشتك بالطبع! سعيدةٌ لمساعدتك في أي وقت. إضغط عليها لترحيبها!

✷ الدليل

حسنًا! فل نشرح المسابقة بشكل أكثر دقة

في كل موسم ينقسم المتنافسين إلى فريقين - يتغير اسم الفرق لكل موسم -

كل فريق يهجم على أعضاء الفريق الآخر برسمه ويسجل بها نقاط لفريقه

الفريق الفائز هو الحاصل على أكبر عدد نقاط في نهاية الموسم.

ملاحظة

ويتم حساب النقاط على الشكل التالي :

  • الرأس لوحده ← 0 نقطة
  • إلى منتصف الجسد ← 0 نقطة
  • كامل الجسد ← 0 نقطة
  • مع خلفية :
    • خلفية بسيطة ← 0 نقطة
    • خلفية معقدة ← 0 نقطة
  • إذا كان رسم تحريك :
    • تحريك بسيط ← 0 نقطة
    • تحريك معقد ← 0 نقطة
  • نوع الرسمة :
    • رسمة بسيطة { تشيبي } ← 0 نقطة
    • رسمة معقدة ← 0 نقطة

ملاحظة أُخرى، قد تزيد نقاطك كلما كانت التفاصيل أكثر { يعتمد على رأي المنسق - مريم أم المشرفين }

✷ القواعد

عند رغبتك في الإلتحاق، هذه هي شروطنا وقواعدنا :

  1. أن تكوني فتاة
  2. أن تكوني فوق ١٢ سنة
  3. أن تكون المشاركات لا تحتوي على ذوات أرواح
    { أي أن ترسمي ملمح واحد أو الأفضل بدون ملامح }
  4. عدم استخدام الذكاء الاصطناعي لأي من الهجمات
  5. أن يكون حسابك خالي من المحرمات
    { الموسيقى، فتيات غير محجبات أو محتوى غير لائق }

إذا لم يتم توفي هذه الشروط أو عدم الإلتزام بها، فلن تلتحق في المسابقة وقد تُطرد من الإلتحاق فور معرفتنا بالمخالفة.
